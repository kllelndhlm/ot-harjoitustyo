# Markdown-editori
Editorilla voi muokata tekstiä markdown-muotoon.

## Käyttöohje
Maalaa ikkunaan kirjoittamasi teksti ja paina painiketta sen mukaan mihin muotoon haluat tekstin. Tekstin voi kopioida leikepöydälle copy-painikkeella.
